# sao-bcd-react-scaffolding

> template to create a react project

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao bcd-react-scaffolding my-project
```

### From git

```bash
sao bryanalx/sao-bcd-react-scaffolding my-project
```
